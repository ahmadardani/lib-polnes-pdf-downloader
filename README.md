## <a name="english"></a>English

Tampermonkey userscript to download hidden PDF blobs from **[lib.polnes.ac.id](https://lib.polnes.ac.id)**. This script intercepts background network requests and provides a download button when a PDF is detected.

### Installation

1. Install the Tampermonkey extension for your browser:
   * **Chrome / Edge / Brave:** [Chrome Web Store](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
   * **Firefox:** [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)

2. Click the link below to automatically install the script:
   * **[Install POLNES Library PDF Downloader](https://raw.githubusercontent.com/ahmadardani/lib-polnes-pdf-downloader/main/lib-polnes-pdf-downloader.user.js)**

### How to Use

1. Go to any document page on **lib.polnes.ac.id**.
2. Wait for the page to load. A green **"Download PDF"** button will appear at the bottom right of your screen once the PDF is detected.
3. Click the button to name and save your file.
4. Press **OK** to save the file to your device.

## <a name="indonesia"></a>Bahasa Indonesia

Userscript Tampermonkey untuk mengunduh PDF blob yang tersembunyi dari **[lib.polnes.ac.id](https://lib.polnes.ac.id)**. Script ini menangkap request jaringan di latar belakang dan memunculkan tombol unduh saat PDF terdeteksi.

### Instalasi

1. Instal ekstensi Tampermonkey di browser kamu:
   * **Chrome / Edge / Brave:** [Chrome Web Store](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
   * **Firefox:** [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)

2. Klik tautan di bawah ini untuk menginstal script secara otomatis:
   * **[Instal POLNES Library PDF Downloader](https://raw.githubusercontent.com/ahmadardani/lib-polnes-pdf-downloader/main/lib-polnes-pdf-downloader.user.js)**

### Cara Penggunaan

1. Buka halaman dokumen apa saja di **lib.polnes.ac.id**.
2. Tunggu halaman selesai dimuat. Tombol hijau **"Download PDF"** akan muncul di pojok kanan bawah layar saat PDF terdeteksi.
3. Klik tombol tersebut, dan kamu akan diminta untuk memasukkan nama file.
4. Tekan **OK** untuk menyimpan file ke perangkat.
